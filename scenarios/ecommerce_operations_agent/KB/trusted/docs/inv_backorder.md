# 제목: 백오더(Backorder) 운영 및 재고 불일치
# 키워드: 백오더, 예약주문, 입고지연, 리드타임, 허수재고
## 백오더 정책
- 재고가 0이어도 입고 일정이 확정된 상품은 주문 가능합니다.
- **필수 조치:** 고객에게 '예상 발송일(리드타임)'을 반드시 사전 안내해야 합니다. (평균 7~10일)

## 재고 불일치 (Phantom Inventory)
- 시스템상 재고가 있으나 실제 창고에 없는 경우, 즉시 주문 취소 및 적립금(2,000원) 보상을 진행합니다.
- `reserved_qty`는 타 고객 결제 진행 중인 수량이므로 주문 불가 수량입니다.
