# 키워드: 분실, 파손, 오배송, 재발송, shipment_lookup
## 분실 의심 (Lost)
- `shipment_lookup` 조회 시 5일 이상 이동 내역이 멈춘 경우 분실로 간주.
- 조치: 택배사 사고 접수 후, 고객 의사에 따라 **선(先) 재발송** 또는 **전액 환불** 진행.

## 파손 및 오배송
- 회사 귀책 사유로 100% 무료 교환/환불 처리.
- 파손된 상품은 사진 1장을 증빙으로 수취 후 처리합니다.
- 오배송 건은 맞교환(회수+재발송)으로 접수합니다.
