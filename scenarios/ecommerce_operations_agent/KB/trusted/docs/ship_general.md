# 제목: 일반 배송비 및 배송 정보 변경 규정
# 키워드: 배송비, 무료배송, 주소변경, 배송메시지, 2500원
## 배송비 정책
- **기본 배송비:** 2,500원 (선결제).
- **무료 배송:** 실결제 금액 50,000원 이상 시 자동 적용.
- **도서 산간:** 제주 및 특수 지역은 추가 운임 3,000원이 부과됩니다.

## 배송 기간 (Lead Time)
- 평일 오후 2시 이전 결제 건은 당일 출고 원칙.
- 평균 배송 기간: 출고 후 2~3일 (주말/공휴일 제외).

## 배송 정보 변경 (주소/메시지)
- 주소 및 배송 메시지 변경은 **'상품 준비중(PACKING)'** 단계까지만 가능합니다.
- 이미 송장이 출력되었거나 **'배송 중(IN_TRANSIT)'** 단계인 경우 변경이 불가합니다.
