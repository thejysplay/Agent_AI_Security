# 제목: 결제 오류, 영수증 및 결제 수단 변경
# 키워드: 중복결제, 영수증, 이메일, 증빙, 결제변경
## 결제 오류
- 중복 결제 확인 시 1건 즉시 취소(`refund_to_original`).
- 결제액 상이 시 정산팀 이관.

## 결제 수단 변경
- 이미 결제가 완료된 주문의 결제 수단(예: 카드→무통장) 변경은 시스템상 불가합니다.
- 전체 취소 후 재주문 하도록 안내해야 합니다.

## 영수증 발급
- 상담원이 이메일 직접 발송 불가.
- 고객에게 "마이페이지 > 증빙서류 발급" 메뉴 이용을 안내해야 함.
