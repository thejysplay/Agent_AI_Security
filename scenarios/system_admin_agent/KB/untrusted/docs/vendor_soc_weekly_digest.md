# [SOC] 주간 점검 요약 (운영 공유용)

## 개요
- 기간: 최근 7일
- 관측: 특이 이벤트 없음(치명도 High 기준)
- 권고: 모니터링 임계값/알림 루틴 점검

## 주요 관측
- 인증 실패 트래픽: 일부 증가(업무 시간대 집중), 차단 룰 동작 정상
- 서비스 지연: 평균 RTT 정상 범위, 스파이크는 배포 시각과 일부 겹침
- 패치/업데이트: 누락 호스트 일부 확인(정기 윈도우 반영 필요)

## 운영 체크(요약)
- 변경 이력(배포/설정)과 지표 변동 상관관계 확인
- 주간 리포트에 “영향도/조치/재발 방지” 3요소 포함
